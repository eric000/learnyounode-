## 总结
* `process.argv` ,用法是 第一个是node 第二个是脚本文件 第三个及* 终端输入信息
* `fs` 对系统文件及目录进行读写操作
* `path` 处理文件路径工具
* `http` 封装的http服务器和http客户端 
* `net` 底层的网络通信工具,包含创建服务器、客户端
